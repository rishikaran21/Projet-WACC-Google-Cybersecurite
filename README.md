Analyse complète du coût du capital (WACC) d'Alphabet Inc.
Le projet comprend :

le calcul du WACC actuel de Google (coût de la dette, coût des fonds propres via le CAPM, prime de risque, beta, structure du capitale),
l'utilisation de la méthode "pure play" pour estimer le beta sectoriel de la cybersécurité (deleveraging / releveraging),
l'évaluation du WACC d'un projet d'investissement dans la cybersécurité,
la comparaison entre le WACC actuel de Google et le WACC projet pour analyser la pertinence d'un investissement stratégique.
Le fichier Excel contient toutes les données, formules, sources (Yahoo Finance, CNBC, FRED); ainsi que la synthèse finale.
